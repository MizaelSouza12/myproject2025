======================================
    WYDBR - Cliente
======================================

Este é o cliente oficial do WYDBR, baseado no jogo original WYD (With Your Destiny)
com melhorias significativas e correções de bugs.

=== REQUISITOS MÍNIMOS DO SISTEMA ===
- Sistema Operacional: Windows 7/8/10/11 (64 bits)
- Processador: Intel Core i3 ou AMD equivalente
- Memória RAM: 4 GB
- Placa de Vídeo: DirectX 9.0c compatível com 1 GB de VRAM
- Espaço em Disco: 2 GB disponíveis
- Conexão com a Internet: Conexão de banda larga (1 Mbps ou superior)

=== INSTALAÇÃO ===
1. Execute o arquivo "WYDBR_Installer.exe" e siga as instruções
2. Após a instalação, inicie o jogo através do ícone criado na área de trabalho

=== CONFIGURAÇÃO ===
Todas as configurações do cliente podem ser ajustadas através do arquivo
WYDBR.ini ou pelo menu de configurações dentro do jogo.

=== ARQUIVOS PRINCIPAIS ===
- WYDBR.exe: Executável principal do jogo
- WYDBR.ini: Arquivo de configuração
- WYDBR_Update.exe: Utilitário de atualização

=== DIRETÓRIOS ===
- Data/: Contém os arquivos de dados do jogo
- Images/: Contém as imagens e texturas
- Sounds/: Contém os arquivos de áudio
- Maps/: Contém os arquivos de mapas
- Effects/: Contém os arquivos de efeitos visuais
- UI/: Contém os arquivos da interface de usuário
- Log/: Contém os arquivos de log
